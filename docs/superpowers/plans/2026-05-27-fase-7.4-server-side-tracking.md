# FASE 7.4 — Server-Side Usage & DLP Tracking

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Tornar o Supabase (`dlp_events`) a fonte da verdade para uso de prompts e eventos DLP — eliminar o estado exclusivamente local em `chrome.storage.local`.

**Architecture:** O backend já grava `dlp_events` para gerações via `/generate-prompts`. A extensão adicionará: (1) um endpoint `POST /auth/dlp/event` para eventos DLP do cliente; (2) `syncUsageFromServer()` que atualiza o cache local após cada geração; (3) `bffTrackDlp()` fire-and-forget que grava scans e proteções no servidor. O `chrome.storage.local` continua como cache rápido para leitura síncrona — nunca como fonte da verdade.

**Tech Stack:** TypeScript (extensão), Python/FastAPI (BFF no VPS), Supabase `dlp_events`, Vitest (testes unitários), `bffFetch()` para autenticação automática com retry.

```typescript
/**
 * Busca contagens reais do servidor e atualiza o cache local.
 * Usa max(local, server) para nunca mostrar valor menor que o já visto.
 * Retorna null se o servidor estiver offline — UI deve usar cache local.
 */
export async function syncUsageFromServer(): Promise<UsageSyncResult | null> {
  const { bffUsage } = await import('../auth/bffClient');
  const server = await bffUsage();
  if (!server) return null;

  // Today
  const localUsage = await getUsage();
  const todayCount = Math.max(localUsage.count, server.today);
  await storageSet({ ...localUsage, count: todayCount });

  // Monthly
  const localMonthly = await getMonthlyUsage();
  const monthlyCount = Math.max(localMonthly, server.monthly);
  const currentMonth = getCurrentMonth();
  await new Promise<void>(resolve => {
    try {
      chrome.storage.local.set(
        { [sk(MONTHLY_KEY)]: { count: monthlyCount, resetMonth: currentMonth } },
        () => resolve(),
      );
    } catch { resolve(); }
  });

  // Total
  const localTotal = await getTotalCount();
  const totalCount = Math.max(localTotal, server.total);
  await new Promise<void>(resolve => {
    try {
      chrome.storage.local.set({ [sk(TOTAL_KEY)]: totalCount }, () => resolve());
    } catch { resolve(); }
  });

  return { todayCount, monthlyCount, totalCount };
}

/** @deprecated Use syncUsageFromServer() */
export async function syncUsageFromSupabase(_jwt: string): Promise<UsageSyncResult | null> {
  return syncUsageFromServer();
}
```
